State Definition Language

Use a SQL-like CASE expression: STATE state FOR LEADING OBJECT TYPE 'Order' AS CASE ... END.

  • WHEN condition THEN 'State label' assigns a state.
  • ELSE 'State label' is optional, but useful for complete coverage.
  • Reference fields with event.type, event.attribute, or object.attribute.
  • Quote spaced names like event."Stock After".
  • Use =, !=, <, <=, >, >=, LIKE, IS NULL, AND, OR, and parentheses.
